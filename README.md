# luxury-fashion-map-nyc

Create a map of luxury fashion and clothing stores in NYC. For each location create a profile based on the website of the brand.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/luxury-fashion-map-nyc.git
cd luxury-fashion-map-nyc
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
